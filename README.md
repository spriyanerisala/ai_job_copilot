AI JOB COPILOT

An AI-powered career assistant that helps job seekers optimize resumes, analyze job descriptions, discover matching jobs, and prepare for interviews using intelligent AI agents built with React.js and n8n.

Overview:
AI Job Copilot is a full-stack AI application designed to simplify the job application process. The platform analyzes resumes against job descriptions, calculates ATS compatibility scores, identifies skill gaps, generates interview questions, recommends matching jobs, and creates role-specific resumes.

The system leverages AI agents orchestrated through n8n workflows, providing an automated and intelligent career guidance experience.


Features : 

1.Resume Analysis
-Upload resume PDF
-Analyze resume against a job description
-Generate ATS compatibility score
-Identify matched skills
-Identify missing skills
-AI-generated resume summary

2.AI Job Matching
-Find jobs matching candidate skills
-Display match percentage
-Explain why a job is suitable
-Highlight matched skills
-Highlight missing skills

3.Interview Preparation
-Generate interview questions automatically
-Categorize questions into:
Easy
Medium
Hard

4.Resume Style Generator
Generate resumes in different formats:
- Fresher Resume
- Internship Resume
- Experienced Resume
- ATS Optimized Resume

5.AI Role-Based Resume Builder
-Analyze uploaded resume
-Suggest suitable job roles
-Generate role-specific resumes
-Improve resume relevance for target positions


6.Modern Responsive UI
-Mobile Responsive
-Tablet Responsive
-Laptop Responsive
-Desktop Responsive
-Built with Tailwind CSS
